#  Análisis del Mercado Bursátil y Proyección para la Toma de Decisiones

##  Objetivo del Proyecto
El objetivo principal de este proyecto es analizar datos históricos del mercado para identificar oportunidades de inversión que provean rendimientos óptimos tanto a corto como a largo plazo, mitigando el riesgo mediante decisiones basadas en datos reales y no en la especulación o el azar.
